Sdl Engine

  • About Project

    • Dsdl-Engine framework is a wrapper framework around SDL liberary. The purpose of the framework is to make it easier to create cross platform games / applications using SDL Media Liberary. My framework includes features for Scene Management, Sprite Rendering, TTF font Rendering, Audio Playback, File I/O, Xml Parsing using Tinyxml-2, Key bindings, Touch Bindings and more.


    The reason I choose to use SDL as a base for my framework is that it provides a low access to graphics hardware via OpenGL, low level access to audio management and keyboard and mouse input giving me a good base to work off. It is already widely used for game development because of the features that it provides to the developer and I taught that I could enhance the use of the library via my framework by suppling a more user friendly experience by providing the user with a higher level of function calls and hiding most of the low level SDL calls the developer usually has to do when using SDL to make games. SDL also has a number of extra libraries that can be incorporated into it for providing extra features for images, audio, and fonts.
